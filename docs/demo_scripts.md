# TOKYO SOUNDS デモ原稿

## 1. オープニング

- これより、『TOKYO SOUNDS』のデモンストレーションを行います。
- チームリーダーのリュウチャーウェイと申します。
- どうぞよろしくお願いします。
- まず、ご覧の通り、
- 画面の中央には、紙飛行機が一機あります。
- ユーザーはこの紙飛行機となり 、
- 東京の街を、
- 思うがままに飛び回ることができます。

## 2. 基本操作

### 2.1 UI

- UI は実際の飛行機を参考に設計されています。
- 画面中央には現在の高さや角度が表示されています。
- 左右には「コンパス」と「速度計」が配置されます。

### 2.2 移動操作

- それでは、実際に動かしてみましょう。
- キーボードの方向キー、または WASD で移動。
- モバイルユーザは画面上のバーチャルコントローラー、
- または姿勢制御も可能です。
- そして、
- SHIFT を押すと……
- このように加速します。

## 3. メニュー

- では、設定画面を見てみましょう。
- 左上のアイコン、
- またはESC キーで開けます。
- ここでは音声の設定や、時間の切り替えが可能です。

- まず、
- ここにご注目ください。
- ここに表示されているのが、
- 「地域パラメーター」です。
- 現在ユーザーの位置が、
- どのエリアの要素を
- どれくらいの割合で
- 含んでいるか、
- その地域の比率に基づいて、
- 現在の時間に加えて、
- AI が音楽を生成する仕組みです。

## 4. 時間変化とUI非表示

- では、時間を「夕方」に切り替えて、
- メニューを閉じてみましょう。
- 新宿のビルたちが夕日（ゆうひ）に染（そ）まり、
- 街の表情が一変しましたね。

- ここで、
- この夕暮れの景色をより楽しみたい方は、
- TAB キーを押してみてください。
- いかがでしょうか。
- 画面上の情報を消すことで、
- より没入感のある飛行が可能になります。

## 5. マルチプレイ

- あっ、あそこにハンターさんが見えてきました！
- そう、
- 実はこの世界には自分だけじゃなく、
- 紙飛行機の周りに他のユーザーも見えるんです。
- さぁ、皆、それでは一緒に原宿へ行きましょう。
- このように、
- 特定のスポットに接近すると、場所の紹介や豆知識が表示されます。

## 6. AI生成音楽と環境音

- この状態で、
- 少し耳を澄（す）ませてみてください。

### 6.1 AI生成音楽

- 聞こえてくるのは、
- 先ほどの地域情報と、
- 現在の時間帯をもとに、
- AI がリアルタイムで生成している音楽です。

### 6.2 環境音

- さらに、よく聞くと、
- 原宿駅のアナウンスが聞こえるはずです。
- これは実際に現地で録音した環境音です。
- そして、
- この環境音は距離に応じて変化する立体音響となります。

## 7. エンディング

- つまり、
- 飛ぶ場所、飛ぶ時間によって、
- 常に一度きりの音声体験が生まれるのです。

- このような、「音」と「映像」で東京を感じる旅。
- 現地の日常音と AI 生成音楽。
- これらを聴きながら、皆と一緒に東京を自由に探索する。
- これこそが、
- 今回の未来創造展のテーマ『ジャポンニズム』に対する、
- 私たちチームの回答、
- 『TOKYO SOUNDS』です。
