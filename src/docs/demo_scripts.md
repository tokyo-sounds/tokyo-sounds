# TOKYO SOUNDS デモ原稿

## 1. オープニング

これより、『TOKYO SOUNDS』のデモンストレーションを行います。
どうぞよろしくお願いします。
まず、ご覧の通り、画面の中央には、[右手伸掌介紹紙飛機]
紙飛行機一機があります。[左手比 1]
ユーザーはこの紙飛行機となり[邊說邊走到右側，接著左手食指指向紙飛機]、
東京の街を、[左手手掌打開伸向飛機下方街道]
思いのままに飛び回ることができます。[左手畫圈，視線朝前看向觀眾]

## 2. UI 紹介と移動操作

### UI

UI は実際の飛行機を参考に設計されています。[邊說邊低頭慢慢走回左側]
画面中央には現在の高さや角度が表示されています。[伸手比向畫面中央]
左右には「コンパス」と「速度計」が配置されます。[伸手比向二側，朝右側走過去介紹速度計]

### 移動操作

それでは、実際に動かしてみましょう。
キーボードの方向キーまたは WASD で移動。
そして、[伸出右手比 1]
SHIFT を押すと…… [做一個右手食指往下按的動作]
このように加速します。[左手伸掌比向畫面中央]

## 3. メニュー

では、設定画面を見てみましょう。
ESC キー、[左手比 1(象徵第一種方法)]
または左上のアイコンでメニューが開きます。[舉左手手指向設定 ICON]
ここでは音声の設定や、時間の切り替えが可能です。[依序比 1,2]

### 音楽パラメータ

まず、ここにご注目ください。[雙手伸手比向選單]
ここに表示されているのが、「地域パラメーター」です。[左手伸手比向標題]
現在ユーザーの位置が、[右手向前五指收攏(象徵現在的位置點)]
どのエリアの要素を
どれくらいの割合で
含んでいるか、[左手在長條圖區域上下滑動]
その比率に基づいて、[右手握拳]
現在の時間も加えて、[左手比向時間選項]
AI が音楽を生成する仕組みです。[雙手伸向前方，掌心朝上]

### 夕方変化

では、時間を「夕方」に切り替えて、[左手指向夕方]
メニューを閉じてみましょう。[慢慢走回左側]
新宿のビルたちが茜色（あかねいろ）に染（そ）まり、[右手伸向畫面]
街の表情が一変しましたね。

## 4. UI 非表示

ここで、[看向觀眾，語氣停頓，雙手向前擺一個停止的動作，微微彎腰引起觀眾注意]
この夕暮れの景色をより楽しみたい方は、TAB キーを押してみてください。[右手比 1]

画面上の情報を消すことで、
より没入感のある飛行が可能になります。[雙手放在兩耳旁，象徵進入沉浸狀態]

## 5. 原宿紹介

あっ、あそこにハンターさんが見えてきました。[右手指向畫面]
そう、実はこの世界には自分だけじゃなく、[右手比 1 擺在臉前]
紙飛行機の周りに他のユーザーも見えるんです。[右手畫一個大圓]
では、一緒に原宿へ行きましょう。[右手指向森林]
特定のスポットに接近すると、このように場所の紹介や豆知識が表示されます。

### AI 音楽と環境音

この状態で、[停頓]
少し耳を澄（す）ませてみてください。[雙手擺在兩耳旁]

#### AI 音楽

聞こえてくるのは、[雙手繼續擺在耳旁]
先ほどの地域解析結果と、[左、右手依序舉起]
現在選択している時間帯をもとに、[雙手握拳]
AI がリアルタイムで生成している音楽です。[雙手打開擺向畫面]

#### 環境音

さらに、よく聞くと、[彎腰低頭右手指向右耳]
原宿駅のアナウンスが聞こえるはずです。[舉起右手比 1]
これは実際に現地で録音した環境音です。[右手指向車站]
そして、この環境音は距離に応じて変化する立体音響です。[雙手在胸前做一個伸縮的手勢]

これらを聴きながら、皆と一緒に東京を自由に探索する。
これこそが、[一邊後退一邊用右手比向畫面]

『TOKYO SOUNDS』です。[雙手打開]
