# TOKYO SOUNDS デモ原稿

## 1. オープニング

これより、私たちチームが開発したサイト、『TOKYO SOUNDS』のデモンストレーションを行います。よろしくお願いします。
まず、ご覧の通り、画面の中央には、紙飛行機一機（いっき）があります。
ユーザーはこの紙飛行機となり、デジタルで再現された東京の街を、思うがままに飛（と）び回ることができます。

## 2. UI 紹介と移動操作

### UI

UI は実際の飛行機を参考に設計しています。画面中央には現在の高さと角度が表示されます。下部（かぶ）の左右（さゆう）には「速度計（そくどけい）」と「コンパス」が配置（はいち）されています。

### 移動操作

それでは、実際に動かしてみましょう。
キーボードの WASD または方向（ほうこう）キーで移動。[動作：A、Dキーを押して左右移動]
スペースキーで一時停止（いちじていし）です。
そして、Shift（しふと）キーを押すと……[動作：Shiftを押して加速]このように加速（かそく）します。

## 3. メニュー

では、設定画面を見てみましょう。ESC（えす）キーでメニューが開きます。 [動作：ESCを押してメニューを開く]
ここでは音声の設定や、時間の切り替え（きりかえ）が可能です。

### 音楽パラメータ

まず、画面の下部（かぶ）にご注目（ちゅうもく）ください。 ここに表示されているのが、**AI 音楽生成の「地域（いき）パラメーター」**です。 現在ユーザーの位置が、どのエリアの要素をどれくらいの割合（わりあい）で含（ふく）んでいるか、その比率（ひりつ）に基（もと）づいて、BGM が合成（ごうせい）される仕組（しく）みです。

### 夕方変化

では、時間を「夕方（ゆうがた）」に切り替え（きりかえ）て、メニューを閉（と）じてみましょう。 [動作：夕方に切り替え->メニューを閉じる]
新宿のビル群（ぐん）が茜色（あかねいろ）に染（そ）まり、街の表情が一変（いっぺん）しましたね。

## 4. UI 非表示

ここで、この夕暮（ゆうぐ）れの景色をより楽しみたい方、キーボードの「Tab（たぶ）キー」を押してみてください。 [動作：TABキーを押してUIを隠す]
いかがでしょうか。
画面上の情報を消すことで、視界（しかい）が広（ひろ）がり、よりシンプルで没入感（ぼつにゅうかん）のある飛行が可能になります。

## 5. 原宿紹介

あそこに「原宿」が見えてきました。近（ちか）づいてみましょう。[動作：右に向かって原宿に接近]
特定のスポットに接近（せっきん）すると、このように場所の紹介や豆知識（まめちしき）が表示されます。

### AI 音楽と環境音

この状態で、少し耳を澄（す）ませてみてください。[動作：原宿駅の上に一時停止]

#### AI 音楽

聞こえてくるのは、先ほどの地域解析（かいせき）と、現在ユーザーが選択している時間帯をもとに、AI がリアルタイムで生成した音楽も流れています。

#### 環境音

さらに、現地で録音した環境音（おん）で、よく聞くと原宿駅のアナウンスが聞こえるはずです。そして、環境音は距離に応（おう）じて音量（おんりょう）が変化する立体（りったい）音響（おんきょう）となっています。

#### 結論

つまり、飛ぶ場所、飛ぶ時間によって、常（つね）に一度きりの音楽体験が生（う）まれるのです。

## 6. エンディング

[動作：加速、上に飛ぶ]
「音」と「映像」で東京を感じる旅。 現地（げんち）の日常音（おん）と、AI が奏（かな）でるテクノロジー音楽。
これらを聴（き）きながら、東京を自由（じゆう）に探索（たんさく）する。
[操作：500m 以上の高さに着いたら、ゆっくりと下（地面）を-5 度の角度で見下ろします]
