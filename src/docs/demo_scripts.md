# TOKYO SOUNDS デモ原稿

## 1. オープニング

これより、『TOKYO SOUNDS』のデモンストレーションを行います。よろしくお願いします。
まず、ご覧の通り、画面の中央には、紙飛行機一機があります。
ユーザーはこの紙飛行機となり、東京の街を、思いのままに飛び回ることができます。

## 2. UI 紹介と移動操作

### UI

UI は実際の飛行機を参考に設計されています。画面中央には現在の高さや角度が表示されています。左右には「コンパス」と「速度計」が配置されます。

### 移動操作

それでは、実際に動かしてみましょう。
キーボードの方向キーまたは WASD で移動。
そして、SHIFT を押すと……
このように加速します。

## 3. メニュー

では、設定画面を見てみましょう。ESC キーでメニューが開きます。
ここでは音声の設定や、時間の切り替えが可能です。

### 音楽パラメータ

まず、ここにご注目ください。
ここに表示されているのが、「地域パラメーター」です。
現在ユーザーの位置が、どのエリアの要素をどれくらいの割合で含んでいるか、その比率に基づいて、AI が音楽を生成する仕組みです。

### 夕方変化

では、時間を「夕方」に切り替えて、メニューを閉じてみましょう。
新宿のビル群が茜色（あかねいろ）に染（そ）まり、街の表情が一変しましたね。

## 4. UI 非表示

ここで、この夕暮れの景色をより楽しみたい方は、TAB キーを押してみてください。
いかがでしょうか。
画面上の情報を消すことで、視界が広がり、よりシンプルで没入感のある飛行が可能になります。

## 5. 原宿紹介

あそこに森が見えてきました。近づいてみましょう。
特定のスポットに接近すると、このように場所の紹介や豆知識が表示されます。

### AI 音楽と環境音

この状態で、少し耳を澄（す）ませてみてください。

#### AI 音楽

聞こえてくるのは、先ほどの地域解析結果と、現在選択している時間帯をもとに、AI がリアルタイムで生成している音楽です。

#### 環境音

さらに、よく聞くと、原宿駅のアナウンスが聞こえるはずです。これは現地で録音した環境音です。
そして、環境音は距離によって変化する立体音響となっています。

## 6. エンディング

つまり、飛ぶ場所、飛ぶ時間によって、常に一度きりの音楽体験が生まれるのです。
このような、「音」と「映像」で東京を感じる旅。
現地の日常音と AI 生成音楽。
これらを聴きながら、東京を自由に探索する。
これこそが、今回の未来創造展のテーマ『ジャポンニズム』に対する、私たちチームの回答、『TOKYO SOUNDS』です。

以上です。ご清聴ありがとうございました。
