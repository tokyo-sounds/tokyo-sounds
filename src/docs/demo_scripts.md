# TOKYO SOUNDS デモ原稿

## 1. オープニング

これより、『TOKYO SOUNDS』のデモンストレーションを行います。よろしくお願いします。
まず、ご覧の通り、画面の中央には、紙飛行機一機（いっき）があります。
ユーザーはこの紙飛行機となり、東京の街を、思うがままに飛（と）び回ることができます。

## 2. UI 紹介と移動操作

### UI

UI は実際の飛行機を参考に設計しています。画面中央には現在の高さと角度が表示されます。左右（さゆう）には「速度計（そくどけい）」と「コンパス」が配置（はいち）されています。

### 移動操作

それでは、実際に動かしてみましょう。
キーボードの方向キーまたはWASDで移動。
スペースキーで一時停止（いちじていし）です。
そして、Shift（しふと）キーを押すと……
このように加速します。

## 3. メニュー

では、設定画面を見てみましょう。ESC（えす）キーでメニューが開きます。
ここでは音声の設定や、時間の切り替えが可能です。

### 音楽パラメータ

まず、画面の下部（かぶ）にご注目ください。 ここに表示されているのが、AI音楽生成の「地域（いき）パラメーター」です。 現在ユーザーの位置が、どのエリアの要素をどれくらいの割合で含（ふく）んでいるか、その比率（ひりつ）に基（もと）づいて、BGM が合成（ごうせい）される仕組みです。

### 夕方変化

では、時間を「夕方（がた）」に切り替えて、メニューを閉（と）じてみましょう。
新宿のビル群（ぐん）が茜色（あかねいろ）に染（そ）まり、街の表情が一変（いっぺん）しましたね。

## 4. UI 非表示

ここで、この夕暮（ゆうぐ）れの景色をより楽しみたい方、「Tabキー」を押してみてください。
いかがでしょうか。
画面上の情報を消すことで、視界（しかい）が広（ひろ）がり、よりシンプルで没入感（ぼつにゅうかん）のある飛行が可能になります。

## 5. 原宿紹介

あそこに森が見えてきました。近づいてみましょう。
特定のスポットに接近すると、このように場所の紹介や豆知識が表示されます。

### AI 音楽と環境音

この状態で、少し耳を澄（す）ませてみてください。

#### AI 音楽

聞こえてくるのは、先ほどの地域解析結果と、現在選択している時間帯をもとに、AI がリアルタイムで生成している音楽です。

#### 環境音

さらに、よく聞くと、原宿駅のアナウンスが聞こえるはずです。これは現地で録音した環境音（おん）です。
そして、環境音は距離に応（おう）じて変化する立体音響となっています。

#### 結論

つまり、飛ぶ場所、飛ぶ時間によって、常（つね）に一度きりの音楽体験が生（う）まれるのです。

## 6. エンディング

「音」と「映像」で東京を感じる旅。
現地の日常音（おん）とAI生成音楽。
これらを聴（き）きながら、東京を自由に探索する。
これこそが、今回未来創造展のテーマ『ジャポンニズム』に対する、私たちチームの回答（かいとう）、『TOKYO SOUNDS』です。
