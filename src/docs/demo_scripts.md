# TOKYO SOUNDS デモ原稿

## オープニング

皆さん、こんにちは。 これより、私たちチームが開発した Web サイト、『TOKYO SOUNDS』のデモンストレーションを行います。
まずは、こちらの画面をご覧ください。 ご覧の通（とお）り、画面（がめん）の中央には一機（いっき）の「紙飛行機」が浮（う）かんでいます。 ユーザーはこの紙飛行機となり、デジタルで再現された東京の街を、思うがままに飛（と）び回ることができます。

## UI 紹介と基本操作

UI は飛行機を参考に設計（せっけい）しています。画面中央には現在の高さと角度が表示されます。画面下の左右（さゆう）には「速度計（そくどけい）」と「コンパス」が配置（はいち）されています。

それでは、実際に動かしてみましょう。 [動作：Wキーを押して移動開始]
キーボードの W、A、S、D で移動。スペースキーで一時停止（いちじていし）です。 そして、Shift キーを押すと……このように加速（かそく）します。 [動作：Shiftを押して加速]

## メニュー・パラメータ・夕方変化

では、設定画面を見てみましょう。ESC キーでメニューが開きます。 [動作：ESCを押してメニューを開く]
ここでは音楽の設定や、時間の切り替え（きりかえ）が可能です。

そして、画面の下部（かぶ）にご注目（ちゅうもく）ください。 ここに表示されているのが、**AI 音楽生成の「地域パラメーター」**です。 現在ユーザーの位置が、どのエリアの要素をどれくらいの割合（わりあい）で含んでいるか、その比率（ひりつ）に基（もと）づいて、BGM が合成（ごうせい）される仕組（しく）みです。

では、時間を「夕方」に切り替え（きりかえ）て、メニューを閉（と）じてみましょう。 [動作：夕方に切り替え->メニューを閉じる]
新宿のビル群（ぐん）が茜色（あかねいろ）に染（そ）まり、街の表情が一変（いっぺん）しましたね。

## UI 非表示 (TAB キー)

ここで、この夕暮（ゆうぐ）れの景色をより楽しみたい方、キーボードの「Tab キー」を押してみてください。 [動作：TABキーを押してUIを隠す]
いかがでしょうか。Tab キーで UI の表示・非表示（ひひょうじ）を切り替えることができます。 画面上の情報を消すことで、視界（しかい）が広がり、よりシンプルで没入感（ぼつにゅうかん）のある飛行が可能になります。

特定（とくてい）のスポットに接近（せっきん）すると、このように場所の紹介や豆知識（まめちしき）が表示されます。

## AI 音楽と環境音

この状態で、少し耳を澄（す）ませてみてください。
聞こえてくるのは、先ほどのパラメーターの解析（かいせき）結果と、現在ユーザーが選択している時間帯をもとに、AI がリアルタイムで生成した音楽も流れています。さらに、実際に録音したリアルな環境音、そして環境音は距離や位置（いち）に応（おう）じて音量（おんりょう）が変化（へんか）する立体音響（りったいおんきょう）となっています。
つまり、飛ぶ場所、飛ぶ時間によって、常（つね）に一度きりの音楽体験が生（う）まれるのです。
