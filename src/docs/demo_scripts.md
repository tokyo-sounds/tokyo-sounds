## 2. UI 紹介と移動操作

### UI

UI は実際の飛行機を参考に設計されています。[邊說邊低頭慢慢走回左側]
画面中央には現在の高さや角度が表示されています。[伸手比向畫面中央]
左右には「コンパス」と「速度計」が配置されます。[伸手比向二側]

### 移動操作

それでは、実際に動かしてみましょう。
キーボードの方向キーまたは WASD で移動。
そして、[伸出右手比1]
SHIFT を押すと…… [做一個右手食指往下按的動作]
このように加速します。[右手伸掌比向畫面中央]

## 3. メニュー

では、設定画面を見てみましょう。
ESC キー、[左手比1(象徵第一種方法)]
または左上のアイコンでメニューが開きます。[走到右側，舉左手手指向設定ICON]
ここでは音声の設定や、時間の切り替えが可能です。

### 音楽パラメータ

まず、ここにご注目ください。[雙手伸手比向選單]
ここに表示されているのが、「地域パラメーター」です。[左手伸手比向標題]
現在ユーザーの位置が、[右手向前五指收攏(象徵現在的位置點)]
どのエリアの要素をどれくらいの割合で含んでいるか、[在長條圖區域上下滑動]
その比率に基づいて、[右手握拳]
AI が音楽を生成する仕組みです。

### 夕方変化

では、時間を「夕方」に切り替えて、
メニューを閉じてみましょう。[慢慢走回左側]
新宿のビルたちが茜色（あかねいろ）に染（そ）まり、街の表情が一変しましたね。

## 5. 原宿紹介

あっ、あそこにハンターさんが見えてきました。。
そう、実はこの世界には自分だけじゃなく、
紙飛行機の周りに他のユーザーも見えるんです。
では、一緒に原宿へ行きましょう。[右手指向森林]
特定のスポットに接近すると、このように場所の紹介や豆知識が表示されます。

### AI 音楽と環境音

この状態で、[停頓]
少し耳を澄（す）ませてみてください。[雙手擺在兩耳旁]

#### AI 音楽

聞こえてくるのは、
先ほどの地域解析結果と、
現在選択している時間帯をもとに、
AI がリアルタイムで生成している音楽です。

#### 環境音

さらに、よく聞くと、[彎腰低頭右手指向右耳]
原宿駅のアナウンスが聞こえるはずです。
これは実際に現地で録音した環境音です。[右手指向車站]
そして、この環境音は距離に応じて変化する立体音響です。

## 6. エンディング
